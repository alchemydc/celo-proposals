## Overview

This proposal requests that the Celo community add Zcash (ZEC) to the basket of currencies used to stabilize the value of Celo stable assets (initially cUSD).

As described in [this post](https://medium.com/celoorg/the-celo-reserve-faqs-f3f7cbb1991f), the present target allocation of the Celo reserve is as follows:
* 50% CELO
* 30% BTC
* 15% ETH
* 5% DAI
